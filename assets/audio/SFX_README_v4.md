# Dead Port Heavy Mech Audio v4

This pack is tuned for a serious large-scale combat machine direction. It is intentionally heavier, lower, longer, and less decorative than v3. No rain audio is included.

## Recommended Phaser timing for 屠灭 / Annihilation

- At warning start: `tumie_colossal_servo_warning_v4`
- During warning buildup: `tumie_reactor_execution_charge_v4`
- On actual slash frame: `tumie_colossal_cut_impact_v4`
- 0.15–0.25 seconds after slash: `tumie_aftershock_metal_collapse_v4`

For impact sounds, avoid stacking too many high-frequency UI sounds at the same time. Let the low-frequency body dominate.
